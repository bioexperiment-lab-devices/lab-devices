# Quickstart

Two short walkthroughs get you from a blank canvas to a finished run. Do them in order.

- **[A. Your first run, no lab needed](a-no-lab.md)** — build a tiny workflow that uses **no
  devices at all**, and run it. The point is to learn the whole loop — build, save, run,
  look at results — without any hardware getting in the way. A few minutes.
- **[B. One pump and one densitometer](b-pump-densitometer.md)** — a real little experiment:
  add dye to a tube of water, watch the optical density climb, and hold a temperature. You'll
  create [roles](../03-concepts/roles.md), map them to real devices, and read the run record.

Each walkthrough builds on the one before. If a term is new, follow its link into
[Concepts](../03-concepts/index.md) and come back.

## What you need

- Experiment Studio open in your browser.
- For **Quickstart A**: nothing else — it runs without a lab.
- For **Quickstart B**: a connected lab with at least one **pump** and one **densitometer**
  (a simulated lab is fine).

Ready? Start with [Quickstart A](a-no-lab.md).
